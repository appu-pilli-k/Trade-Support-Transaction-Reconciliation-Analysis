# Trade-Support-Transaction-Reconciliation-Analysis
Project Overview

This project focuses on transaction reconciliation and exception monitoring using SQL and Power BI concepts. The objective was to identify operational exceptions, monitor settlement performance, and generate KPI-based insights for trade support operations.

Transaction and settlement reconciliation

Operational KPI reporting
